## Description

 NestJs Boilerplate

## Installation

```bash
$ npm i
```

## Migration

```bash
$ npm run typeorm:migrate
```

## Seed

```bash
$ npm run typeorm:seed:generate
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
